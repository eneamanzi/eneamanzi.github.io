Developed a multi-threaded FTP-compliant client-server application in **C** for file transfer, fully interoperable with FileZilla. Course Project (A.Y. 2022-2023).

### Technical Highlights:
* **Socket Programming:** Architecture based on low-level TCP connections for concurrent session management.
* **Data Modes:** Native support for both **Active (PORT)** and **Passive (PASV)** data transfer modes.
* **Protocol Management:** Implementation of essential FTP commands (e.g., USER, PASS, LIST, RETR, STOR, CWD).
* **Performance:** Optimized for reliable file transfer and multi-threaded execution.
